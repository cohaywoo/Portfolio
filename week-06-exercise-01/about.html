About This Navigation Lab

Purpose

This exercise focuses on building a multi-page website with proper linking architecture and experimenting with different navigation styles. It's designed to help you understand how websites are structured across multiple HTML files and folders.

The Minecraft Theme

We've styled this exercise with a Minecraft aesthetic to make learning about web architecture more fun and memorable. The blocky design, earthy color palette, and familiar Minecraft elements help reinforce the concept of "building" a website structure.

Navigation Pattern: Stone Steps

The navigation on this page uses our second pattern - "Stone Steps". This style creates a 3D effect with inset shadows, making each link look like a raised stone block. Hover over the links to see the lifted effect with the diamond-colored highlight.

What Makes This Lab Valuable

  • Relative Paths: You'll practice using ../ to navigate between folder levels
  • CSS Linking: Nested pages must link to the shared CSS file correctly
  • Navigation Consistency: Different patterns on different pages, all pointing to the right places
  • CSS Creativity: Designing and styling multiple navigation approaches

Continue Exploring

Click on "Projects" or "Gallery" in the navigation to see how the links work from nested folders. Notice that those pages use different navigation styles and have different link syntax because they're inside subdirectories.
