Welcome to the Minecraft Navigation Lab!

This is your home page in our Minecraft-themed navigation experiment. We're practicing multi-page website architecture and exploring different navigation styles.

In this lab, you'll explore three different navigation patterns across our site, each styled with a Minecraft aesthetic featuring blocky elements, pixel-perfect borders, and Minecraft color palettes.

What You'll Learn

• Understanding relative file paths for multi-page websites
• Implementing different navigation patterns with CSS
• Linking pages within nested folder structures
• Creating hover effects and interactive elements
• Building a Minecraft-themed design system

Navigation Patterns

This page (Home): Uses the "Blocky" navigation pattern - a wood-themed horizontal menu
About page: Uses the "Stone Steps" navigation pattern - with a 3D stepped effect
Projects page: Uses the "Diamond Luxury" navigation pattern - centered with diamond styling
Gallery page: Also uses the "Diamond Luxury" pattern

Explore the Site

Use the navigation menu above to explore each section of the site. Pay attention to how the links change based on your location in the folder structure. Notice how nested pages use relative paths (../) to navigate back to the root directory.
