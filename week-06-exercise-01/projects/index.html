Building Blocks of Web Architecture

This is the Projects page, located inside the projects/ folder. Notice that this page uses the "Diamond Luxury" navigation style - the third pattern we're experimenting with.

Notice the Paths

Because this page is nested inside the projects/ folder, the navigation links use different syntax:

  • ../index.html - goes up one level to reach Home
  • ../about.html - goes up one level to reach About
  • index.html - stays in the current projects folder
  • ../gallery/ - goes up one level, then into gallery
  • ../css/style.css - CSS link also uses ../ to go up one level

Available Projects

Navigate to our project detail page to see a nested page within this folder. Notice how the path changes when you're linking to another file in the same directory.

Flexbox in Action

All of our navigation patterns use Flexbox to create responsive layouts. The "Diamond Luxury" pattern on this page uses justify-content: center to center the navigation items, and the responsive design adapts on smaller screens.
