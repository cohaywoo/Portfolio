Project Detail: Understanding Relative Paths

Deeply Nested Content

This page is located at projects/project-detail.html - it's inside the projects folder, just like index.html. Both files use the same relative paths to access shared resources.

CSS Path Lesson

Notice the CSS link in the HTML head: <link rel="stylesheet" href="../css/style.css">

The ../ means "go up one level from the current directory". From projects/, we go up to the root, then down into the css/ folder. This is essential for nested pages to access the shared stylesheet.

Navigation Consistency

Even though this is a different page within the projects folder, it uses the same navigation pattern as the projects index. This creates a consistent experience throughout the site.
